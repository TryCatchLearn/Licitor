# Challenge 2

## What you see

On a Draft listing you own, you click **Set as main** on one of your secondary images. The button spins briefly and then resets — no error message appears. Everything looks fine.

But when you reload the page, the listing has **no main image**. The listing card in the public marketplace also shows a broken or missing image placeholder.

## How to reproduce

1. Create a new listing and upload **2 or more images**
2. On the listing detail page, click **Set as main** on any secondary (non-main) image
3. Observe the button completes without error — the UI updates locally
4. Reload the page
5. The listing now displays no main image
