# Challenge 1

## What you see

You create a fresh listing and try to save your edits using the "Refine listing" dialog. The form submits but you get this error:

Listings with bids can no longer be edited.

The listing was just created and has never received a bid. Refreshing and trying again gives the same result.

## How to reproduce

1. Create a new listing (upload an image → Continue)
2. On the listing detail page, click Refine listing
3. Fill in any valid title, description, dates, and prices
4. Click Save — observe the error
