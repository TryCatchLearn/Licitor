# Challenge 4

## What you see

You publish a listing and the action completes without error. The UI refreshes. But your listing continues to appear in **My Listings** showing a **Draft** status — across every tab (Draft, Active, Scheduled). Other users cannot find it in the public marketplace.

You investigate, find what looks like the problem, apply a fix — and now the listing **vanishes from My Listings entirely**. Yet it now appears in the public marketplace.

## How to reproduce

1. Create a new listing
2. Click **Publish now**
3. Observe: no error, page refreshes
4. Navigate to **My Listings** → listing shows under Draft tab
5. Click the Active tab → listing also shows there
6. Check the public listings page → listing is absent
